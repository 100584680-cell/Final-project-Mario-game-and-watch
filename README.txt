1. DEPLOYMENT INSTRUCTIONS
========================================================================
To ensure the software runs correctly, please follow these steps strictly 
in the order presented.

STEP 1: UNARCHIVE
-----------------
Extract (Unzip) the contents of the compressed file to a local directory 
on your computer.


STEP 2: ENVIRONMENT SETUP
-------------------------
This application requires Python 3.x and the 'Pyxel' library.

If pyxel is not installed:
1. Open your terminal (Command Prompt/PowerShell on Windows or 
   Terminal on macOS/Linux).

2. Execute the following command to install the required dependency:
   
   pip install pyxel

   (Note: If the command above fails, try: pip3 install pyxel)

STEP 3: EXECUTION
-----------------
Please choose ONE of the following methods to start the game.

[ OPTION A: RUNNING VIA TERMINAL / COMMAND LINE ]
-------------------------------------------------
1. Open your terminal.

2. Navigate to the extracted project folder using the 'cd' command.
   Example:
   cd C:\Users\YourName\Desktop\Final-project-Mario-game-and-watch

3. Run the application by executing the entry point script:

   python main.py


[ OPTION B: RUNNING VIA VISUAL STUDIO CODE (VS CODE) or pycharm]
------------------------------------------------------
1. Launch Visual Studio Code.

2. Go to "File" -> "Open Folder..."

3. Select the folder where you extracted the project files. 
   IMPORTANT: You must open the entire FOLDER, not just the 'main.py' 
   file. This ensures the game can locate the 'assets.pyxres' file.

4. Once the folder is loaded in the Explorer pane, click on 'main.py' 
   to open it.

5. Click the "Run Python File" button (Play icon) in the top right 
   corner, or press F5 to start debugging.

========================================================================
2. CONTROL SCHEME
========================================================================
The application accepts the following user inputs:

[ UP / DOWN ARROWS ] : Move Mario
[ W / S ]   : Move Luigi
[ Q ]                   : Quit Application

Project can be also found at https://github.com/100584680-cell/Final-project-Mario-game-and-watch
